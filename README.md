# Requirements
  ```sh
  node.js v18 or later
  ```

# Install & Run
  ```sh
  npm install
  npm run dev
  ```
Example request:
```sh
curl http://localhost:8000/api/v1/delivery-order-price?venue_slug=home-assignment-venue-helsinki&cart_value=1000&user_lat=60.17094&user_lon=24.93087
```
# Run tests
  ```sh
  npm run test
  ```
